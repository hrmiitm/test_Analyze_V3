Analyze_V3

This repository contains a small data analysis script (execute.py) that reads data.csv and writes result.json.

The GitHub Actions workflow (.github/workflows/ci.yml) runs ruff, executes the script and publishes the generated result.json via GitHub Pages.

When the workflow runs, the generated result will be available at /result.json.

If you want to view results locally, run:

python execute.py > result.json
python -m http.server 8000
# then open http://localhost:8000/result.json
